# 🚀 Poor AI - The Ultimate Coding Assistant

Welcome to **Poor AI**, a professional, highly capable programming assistant and AI chat platform. Built for developers, by developers. 

## ✨ Key Features
* **Universal API Engine:** Powered by OpenRouter, seamlessly routing to the best AI models (Gemini Pro, GPT-3.5, Claude).
* **Pro-Level Code Generation:** Dark-themed code blocks with syntax highlighting and instant "Copy Code" functionality.
* **Multimodal Input:** Supports Voice Input (Speech-to-Text) and Image Uploads (Base64).
* **Secure Cloud Storage:** Real-time user authentication and chat history saved via Google Firebase.
* **Master Admin Dashboard:** A secret control panel for the Admin to manage users, track stats, and grant "Pro" access.
* **Automated Payments:** Manual payment gateway integrated directly with WhatsApp for instant communication.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript.
* **Backend/Database:** Firebase Authentication & Cloud Firestore.
* **API Security Proxy:** Google Apps Script (Hides the API Key completely).
* **Hosting:** GitHub Pages (Lifetime Free).

## 💡 How the Pro Subscription Works
Users can upgrade to Pro to unlock unlimited code generation and premium AI models. Payments are handled manually via local mobile banking (bKash/Nagad), and the Admin can approve Pro access with a single click from the Admin Panel.
